Data Entry Project: Contact Details from PDF

Description

This project involved extracting names, titles, and contact details from a provided PDF document and entering them into a structured format in either a CSV or XLSX spreadsheet file.

Task

To accurately gather and organize the following details for each contact:

  • Name
  • Title
  • Company
  • Street Address
  • City
  • State
  • ZIP
  • Phone
  • Fax
  • Email
  • Website (if available)

Resource File

The details were extracted from a PDF file, which can be found here.

Solution

  • Opened and reviewed the PDF document to locate all relevant details.
