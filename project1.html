Data Entry Project: Scanned PDF to Word Conversion

Description

The client provided a scanned PDF files, which needed to be converted into Microsoft Word documents. The task involved accurately transferring the text, maintaining the exact formatting, and ensuring that any footer or background color was preserved in the final document.

Task

The objective was to convert the scanned PDFs into Word documents while retaining all the original formatting. Specifically, the font needed to be set to Arial, size 11, and any background color and footer information had to be included exactly as it appeared in the source files.

Solution

I used a combination of manual data entry and automated tools to convert the PDFs into Word documents. Here’s a breakdown of the process:

  • Downloaded and reviewed the provided PDFs to understand the content layout.
  • Used Optical Character Recognition (OCR) software to extract the text from the scanned files.
  • Manually adjusted the formatting in Microsoft Word to match the original documents, ensuring font type and size, text alignment, and footer information were accurate.
  • Used Microsoft Word’s styles and formatting options to apply the correct fonts, sizes, and colors.
